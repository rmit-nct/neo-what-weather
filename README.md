## üöÄ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm, pnpm, or bun
- OpenWeather API key

### Installation

1. Clone the repository
```bash
git clone https://github.com/rmit-nct/neo-what-weather.git
cd neo-what-weather
```

2. Install dependencies
```
npm install
```

3. Create env file
```
Add an .env file to the root folder:
VITE_API_KEY = <YOUR_OPEN_WEATHER_API_KEY>
```

4. Start the development server
```
npm run dev
```
